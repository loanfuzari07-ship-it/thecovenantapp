# **App Name**: The Covenant App

## Core Features:

- User Session Management: Enable secure user login via email and manage session persistence using the browser's localStorage, entirely frontend-driven without a backend.
- 21-Day Protocol Progress Tracking: Provide access to and tracking of 21 daily guided audio prayers, allowing users to mark each day as complete with progress stored in localStorage.
- Daily Devotional Readings & Note-taking: Present 21 daily devotionals with scripture and reflection text, coupled with an integrated system for users to write, save, and retrieve personal notes for each day in localStorage.
- Generative Reflection Assistant: An AI tool that processes a user's stored devotional notes and protocol progress to generate personalized insights, encouraging deeper self-reflection, or suggesting custom prayer prompts.
- Comprehensive Journey Progress Display: Visually represent the user's overall advancement through the 21-day program, including separate progress for protocol days and devotionals, on the home dashboard.
- Supplemental Bonus Content Access: Deliver a curated collection of bonus materials, such as Sacred Frequency Audios with detailed guidance and practical advice on crafting supportive text messages for marital communication.
- Direct Support System: Facilitate direct user support by providing clear contact information, including email, operational hours, and a pre-configured email client link for inquiries.

## Style Guidelines:

- Primary color: A rich, golden hue (#CEAC6E), evoking divinity, commitment, and spiritual warmth. This choice provides a luminous contrast against the deep background, guiding the user's focus through the content.
- Background color: A profound, earthy brown (#241F19), establishing a calm, reverent, and deeply focused atmosphere suitable for personal reflection and spiritual journeying.
- Accent color: A vibrant red-orange (#D85C17), offering an energetic and purposeful highlight. This color will be used sparingly for critical interactive elements and status indicators to ensure clarity and impact.
- Headline font: 'Lora' (serif), selected for its elegant, literary, and slightly formal characteristics, reinforcing the sacred and introspective nature of the application's content. Body text font: 'Inter' (sans-serif), chosen for its exceptional readability, modern appearance, and neutrality, ensuring a comfortable reading experience for extended reflection texts. Note: currently only Google Fonts are supported.
- Minimalist and clean SVG path icons for navigation and key content elements. This approach ensures visual consistency, fast loading, and scalability across mobile devices, contributing to a streamlined user experience.
- A responsive, mobile-first design centered on a maximum width of 480px. Features include a fixed header for branding and a persistent bottom navigation bar for effortless tab switching, ensuring core functions are always accessible. Content areas dynamically display with smooth opacity fade transitions.
- Subtle and purposeful animations will be employed for view transitions (e.g., fading content when switching tabs) and for interactive components like expanding content cards, enhancing engagement without distracting from the app's core purpose.
